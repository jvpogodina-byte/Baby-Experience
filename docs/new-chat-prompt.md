# Новый чат: продолжение плана

Продолжаем обсуждение плана проекта рекомендаций для мамы и новорожденного.

Контекст:
- Full-stack веб-приложение с публичной частью и админкой.
- Публичная часть показывает рекомендации по категориям, карточки вещей, объяснения, примеры конкретных вещей, комментарии и сохранённые списки.
- Авторизация нужна для комментариев и сохранения списков.
- В MVP используем Next.js + PostgreSQL + Prisma + Auth.js/NextAuth.
- Поддерживаем email/password и Google OAuth.
- Нужен `draft -> published` workflow в админке.

Что нужно обсудить дальше:
- детальную структуру страниц и роутов,
- схему данных и поля сущностей,
- поток авторизации и роли,
- сценарии админки,
- UI/UX для карточек вещей и списков,
- этапы MVP и последующие улучшения.
